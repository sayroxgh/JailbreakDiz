# CS2 Jailbreak Diz Plugin

A CounterStrikeSharp plugin for Counter-Strike 2 Jailbreak servers.

## Features
- Line up all Terrorist players automatically
- Select lineup positions by shooting two points
- Freeze players in place
- Break lineup with command
- Automatic cleanup on round end

## Commands
- `css_diz` → Start lineup setup
- `css_dizboz` → Break the lineup

## Requirements
- Metamod
- CounterStrikeSharp
- CS2 Dedicated Server
